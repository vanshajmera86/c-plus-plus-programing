
	Inheritance : It is acquiring the properties of base class into the 
		      derived class.
	
	practical definition : 	It is creating a new class from an already 
				existing class.

	Important note.	     :  Objects will always be created of Derived Class 
				in Inheritance.	


	Features	     : Code Reusability.
			        No Duplicacy of Code.


	Types of Inheritance :

	There are 5 types of Inheritance in C++.

	1) Single-level Inheritance
	2) Multi-level Inheritance
	3) Multiple level Inheritance
	4) Hierarchical level Inheritance
	5) Hybrid level Inheritance.
